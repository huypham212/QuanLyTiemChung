# Quản Lý Tiêm Chủng
## Description
This is graduation project. 
## Pre-Required
**Node.js:** `8.3 or latest`  
**React Native:** `0.67 or latest`  
**AngularCLI:** `13.0.0 or latest`  
## Usage
### Run on Mobile
On root project, run:  
```sh 
cd Mobile\tiemchhung
```
```sh 
npm install
```  
```sh
npm react-native run-android
```
### Run on Web
On root project, run:  
```sh 
cd BE\tiemchung
```
```sh 
npm install
```  
```sh
npm start
```
